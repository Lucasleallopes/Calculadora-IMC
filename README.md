# Calculadora de IMC

Este projeto é uma Calculadora de Índice de Massa Corporal (IMC) simples, construída com HTML, CSS e JavaScript. Ela permite que os usuários insiram sua altura e peso para calcular seu IMC e entender em qual categoria de peso eles se enquadram.

## Recursos

- **Interface Simples**: Uma interface de usuário limpa e amigável que facilita a entrada de dados.
  
- **Cálculo Instantâneo**: O IMC é calculado instantaneamente após a entrada de altura e peso, fornecendo feedback imediato ao usuári.
  
- **Feedback Visual**: A cor de fundo da página muda de acordo com a categoria do IMC, proporcionando uma representação visual intuitiva do resultado.
